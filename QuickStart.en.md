Quick Start
---
ISEKAI Computation is a computation platform that enables secure data processing.

A data analyst (data processing operator) sends a request describing the computation to execute on ISEKAI Computation, and the computation is then run. The data to be processed is transmitted from ISEKAI Data Server to ISEKAI Computation.

Let's try it now. Access [JupyterLite for ISEKAI Computation](https://seera-networks.github.io/jupyterlite/), open `QuickStart.ipynb`, and follow the instructions to run the notebook.

For `data_token`, specify the following string:
```
CqRjmM2HHO
```

A detailed explanation of the DAG used to describe computations is available in [Dag Reference](dag-reference.md).
